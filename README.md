# Trojan-WebUI
为Trojan多用户模式打造的管理工具


## 现在已经实现了基本功能，包括流量的管理，充值，服务器管理，获取，最下边是演示demo和一些演示图片.
# 1.部署教程
下载发行版本；
导入数据库；
克隆项目，导入到idea；
在BaseDao类中更改数据库的连接信息
在js/html.js中更改tomcat的url；
编译完成后即可运行；
## 注意事项
